# Quiz-Game
Simple and Advanced Quiz Game

Simple Quiz Game Features:
- Hard-coded questions and answers into an array.
- Loops through the questions until end.
- Shows the total score at the end.

Advanced Quiz Game Features:
- Checks the project for quiz files.
- Quiz topic chooser.
- Quizzes are stored in files.
- Title and total items based on the actual quiz chosen.
- Loops through the questions until end.
- Shows the total score at the end.
- Option to view the answer key.
